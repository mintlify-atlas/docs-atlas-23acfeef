The X-axis is a horizontal axis in Cartesian coordinate charts. It works with the grid component to position and render data along the horizontal dimension.

## Basic Configuration

The X-axis supports four axis types: `value`, `category`, `time`, and `log`. The type determines how data is scaled and displayed.

```javascript
option = {
  xAxis: {
    type: 'category',
    data: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun']
  },
  yAxis: {
    type: 'value'
  },
  series: [{
    data: [120, 200, 150, 80, 70, 110, 130],
    type: 'bar'
  }]
}
```

## Axis Types

### Value Axis

Numerical axis suitable for continuous data. Automatically calculates min, max, and interval values.

```javascript
xAxis: {
  type: 'value',
  min: 0,
  max: 100,
  splitNumber: 5
}
```

Reference: [src/coord/axisCommonTypes.ts:207-217](~/workspace/source/src/coord/axisCommonTypes.ts)

### Category Axis

Ordinal axis for discrete categorical data. Requires explicit data definition.

```javascript
xAxis: {
  type: 'category',
  data: ['Q1', 'Q2', 'Q3', 'Q4'],
  boundaryGap: true,
  axisTick: {
    alignWithLabel: true
  }
}
```

Reference: [src/coord/axisCommonTypes.ts:182-206](~/workspace/source/src/coord/axisCommonTypes.ts)

### Time Axis

Temporal axis optimized for time-series data with automatic time formatting.

```javascript
xAxis: {
  type: 'time',
  splitNumber: 10,
  axisLabel: {
    formatter: '{yyyy}-{MM}-{dd}'
  }
}
```

Reference: [src/coord/axisCommonTypes.ts:223-226](~/workspace/source/src/coord/axisCommonTypes.ts)

### Log Axis

Logarithmic axis for exponential data patterns.

```javascript
xAxis: {
  type: 'log',
  logBase: 10,
  min: 1,
  max: 100000
}
```

## Advanced Examples

### Multiple X-Axes

```javascript
option = {
  xAxis: [
    {
      type: 'category',
      data: ['A', 'B', 'C', 'D'],
      position: 'bottom'
    },
    {
      type: 'category',
      data: ['Category 1', 'Category 2', 'Category 3', 'Category 4'],
      position: 'top'
    }
  ],
  yAxis: {
    type: 'value'
  },
  series: [{
    data: [10, 20, 30, 40],
    type: 'bar',
    xAxisIndex: 0
  }]
}
```

### Time Axis with Custom Formatting

```javascript
xAxis: {
  type: 'time',
  axisLabel: {
    formatter: {
      year: '{yyyy}',
      month: '{MMM}',
      day: '{d}',
      hour: '{HH}:{mm}',
      minute: '{HH}:{mm}'
    }
  },
  splitNumber: 10
}
```

### Value Axis with Custom Range

```javascript
xAxis: {
  type: 'value',
  min: function(value) {
    return value.min - 10;
  },
  max: function(value) {
    return value.max + 10;
  },
  scale: true,
  boundaryGap: ['10%', '10%']
}
```
