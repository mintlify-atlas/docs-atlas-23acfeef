# TypeScript Type Definitions

Apache ECharts provides comprehensive TypeScript type definitions for type-safe chart development.

## Importing Types

### From NPM Package

```typescript
import * as echarts from 'echarts';
import type {
  EChartsOption,
  SeriesOption,
  LineSeriesOption,
  BarSeriesOption
} from 'echarts';
```

### Direct Import from Types

```typescript
// Import main option type
```

## Using Types in Your Code

### Basic Example

```typescript
import * as echarts from 'echarts';
import type { EChartsOption } from 'echarts';

const option: EChartsOption = {
  title: {
    text: 'Sales Data'
  },
  xAxis: {
    type: 'category',
    data: ['Mon', 'Tue', 'Wed']
  },
  yAxis: {
    type: 'value'
  },
  series: [
    {
      type: 'line',
      data: [120, 200, 150]
    }
  ]
};

const chart = echarts.init(document.getElementById('main'));
chart.setOption(option);
```

### Strict Type Example

```typescript
import type { LineSeriesOption, BarSeriesOption } from 'echarts';

const lineSeries: LineSeriesOption = {
  type: 'line',
  name: 'Temperature',

  smooth: true,
```
